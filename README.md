# Kayson Wheels - React Website

This is the React version of the Kayson Wheels website, converted from static HTML.

## Features

- Modern React application built with Vite
- React Router for navigation
- Responsive design
- All original styles and functionality preserved

## Pages

- Home
- About Us
- Auto Chains
- Motorcycle Sprockets
- Quality Policy
- Contact Us
- Enquiry Form

## Installation

```bash
npm install
```

## Development

To run the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

To build for production:

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
